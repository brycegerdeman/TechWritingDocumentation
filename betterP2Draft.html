Adding SSH Extensions

Step 1. Look to menu bar on the left side

Step 2. Go to the fifth icon down

Step 3. Hover your cursor over the extension icon

Step 4. Click on the extension icon

Step 5. Click the search bar

Step 6. Type “Remote - SSH”

Step 7. Click on Remote - SSH extension

Step 8. Click Install Button

How to Make a Checkstyle XML File

Step 1. Right click on the leftmost panel

Step 2. Select "New File" from the menu

Step 3. Name your file with the filename and .xml tag

Step 4. Double click on the file to open it

Step 5. Type in the version number as the first line

This will typically be version 1 as you have just created the file

Step 6. On a new line, insert the link to puppycrawl as the doc type

The insertion is as follows for your convenience: "-//Puppy Crawl//DTD Check Configuration 1.2//EN" "http://www.puppycrawl.com/dtds/configuration_1_2.dtd"

Step 7. Type in the general module name as "Checker"

This is so that the processor knows the file is a checkstyle file

Step 8. Press enter twice

Step 9. Press tab to insert new checks within the module

Step 10. Insert your first rule as a module named LineLength

Step 11. On a new line, enter the name of the check as "max"

"max" is the maximum number of chracters on a line

Step 12. Enter the value as the number of characters you would like to constrain to

In this example, 80 characters is the maximum

Step 13. On a new line, enter "/module" in arrows to finish the rule

Step 14. On the last line of the file, enter another "/module" in arrows

This will close the checkstyle document

Step 15. Visit the puppycrawl website to see which other checks you would like to implement

https://checkstyle.sourceforge.io/checks.html

How to connect VSCode to the Student Machine using SSH

Step 1: Click the blue button on the bottom left corner

You can also enter control shift p

Step 2: Enter the following command:

“Remote-SSH: Add New SSH Host…”

Step 3: Click “ + Add New SSH Host”

Step 4: Enter “ssh USERNAME@student2.cs.appstate.edu”

Replace USERNAME with your username.

Step 5: Select Linux

Fun Fact: The student machine is Linux.

Step 6: Press Enter

Step 7: Click the “Connect” button

The connect button is in the bottom right corner

Step 8: Enter your password:

Visit the FAQ page for common passwords if you don't know yours!

FAQ

Q: What is SSH?
A: SSH is a secure protocol used as the primary means of connecting to Linux servers remotely.

Q: What are some helpful extensions?
A: It depends on what you're working on! Each language has its own extension which is extremely useful. Development environments for languages like HTML are also very useful.

Q: Why do I choose Linux for the connection type?
A: This is because that's the OS that is run on the student2 servers, as well as most of the other servers we have on campus.

Q: Can I do this in the terminal?
A: You definitely can! You just type "ssh yourusername@student2.cs.appstate.edu"
