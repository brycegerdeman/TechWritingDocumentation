How to Make a Checkstyle XML File

1. Right click on the leftmost panel

2. Select "New File" from the menu

3. Name your file with the filename and .xml tag

4. Double click on the file to open it

5. Type in the version number as the first line

6. Insert the link to puppycrawl as the doc type

The insertion is as follows for your convenience: "-//Puppy Crawl//DTD Check Configuration 1.2//EN" "http://www.puppycrawl.com/dtds/configuration_1_2.dtd"

7. Type in the general module name as "Checker"

This is so that the processor knows the file is a checkstyle file

8. Press enter

9. Press tab to insert new checks within the module

10. Insert your first rule as a module named LineLength

11. Include a severity of error

12. Include a value to the severity

13. Visit the puppycrawl website to see which other checks you would like to implement

https://checkstyle.sourceforge.io/checks.html

How to connect VSCode to the Student Machine using SSH

Step 1

Click the blue button on the bottom left corner

Step 2

Enter the following command: “Remote-SSH: Add New SSH Host…”

Step 3

Click “+ Add New SSH Host”

Step 4

Step 5

Select “Linux”

Step 6

Press enter

Step 7

Click the “Connect” button in the bottom right corner

Enter your password
