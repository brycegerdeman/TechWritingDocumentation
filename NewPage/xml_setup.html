How to Make a Checkstyle XML File

Step 1. Right click on the leftmost panel

Step 2. Select "New File" from the menu

Step 3. Name your file with the filename and .xml tag

Step 4. Double click on the file to open it

Step 5. Type in the version number as the first line

This will typically be version 1 as you have just created the file

Step 6. On a new line, insert the link to puppycrawl as the doc type

The insertion is as follows for your convenience: "-//Puppy Crawl//DTD Check Configuration 1.2//EN" "http://www.puppycrawl.com/dtds/configuration_1_2.dtd"

Step 7. Type in the general module name as "Checker"

This is so that the processor knows the file is a checkstyle file

Step 8. Press enter twice

Step 9. Press tab to insert new checks within the module

Step 10. Insert your first rule as a module named LineLength

Step 11. On a new line, enter the name of the check as "max"

"max" is the maximum number of chracters on a line

Step 12. Enter the value as the number of characters you would like to constrain to

In this example, 80 characters is the maximum

Step 13. On a new line, enter "/module" in arrows to finish the rule

Step 14. On the last line of the file, enter another "/module" in arrows

This will close the checkstyle document

Step 15. Visit the puppycrawl website to see which other checks you would like to implement

https://checkstyle.sourceforge.io/checks.html
