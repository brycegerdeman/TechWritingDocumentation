How to Make a Checkstyle XML File

1. Right click on the leftmost panel

2. Select "New File" from the menu

3. Name your file with the filename and .xml tag

4. Double click on the file to open it

5. Type in the version number as the first line

6. Insert the link to puppycrawl as the doc type

7. Type in the general module name as Checker so that the processor knows this is a checkstyle file

8. Press enter

9. Press tab to insert new checks within the module

10. Type in the module name as TreeWalker because this is the most common module used

11. Press enter

12. Press tab to insert a new rule within TreeWalker

13. Insert your first rule as another module named LineLength

14. Include a severity of error if you wish

15. Include a value to the severity if you included it

16. Visit the puppycrawl website to see which checks you would like to implement https://checkstyle.sourceforge.io/checks.html
